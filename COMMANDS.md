# COPY PASTE COMMANDS

Pilih command sesuai kebutuhan dan copy-paste ke terminal.

## Scenario 1: Update index.html

```bash
python3 auto-push.py "masukan file index.html"
```

## Scenario 2: Update CSS

```bash
python3 auto-push.py "masukan file style css"
```

## Scenario 3: Multiple Files

```bash
python3 auto-push.py "masukan file index.html style css intinga file yang ada"
```

## Scenario 4: Backend Update

```bash
python3 auto-push.py "Update backend dan database fitur"
```

## Scenario 5: Setup Features

```bash
python3 auto-push.py "Tambah setup, biaya, bandwidth, backend sections"
```

## Scenario 6: General Update

```bash
python3 auto-push.py "Update website dengan konten baru"
```

## Scenario 7: Initial Commit (First Time)

```bash
python3 auto-push.py "Initial website commit"
```

---

## Jika Menggunakan Bash Script

Ganti `python3 auto-push.py` dengan `bash auto-push.sh`:

```bash
bash auto-push.sh "masukan file index.html"
```

---

## Jika Ingin Manual

```bash
git add .
git commit -m "masukan file index.html"
git push origin main
```

---

## Tips

- Ubah pesan sesuai dengan file/fitur yang Anda update
- Script akan otomatis detect file yang berubah
- Tidak perlu mengetik path file secara manual
- Website akan otomatis terupdate di GitHub Pages
