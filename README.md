## 🚀 Cara Menggunakan

### Untuk Development Lokal:
```bash
# Clone repository
git clone https://github.com/najibalwafa/website-najibalwafa.git
cd website-najibalwafa

# Buka di browser (bisa langsung buka index.html)
# Atau gunakan HTTP server:
python -m http.server 8000
# Kunjungi: http://localhost:8000
```

### Deploy ke GitHub Pages:
```bash
# Pastikan repository bernama: username.github.io
git add .
git commit -m "Update website"
git push origin main
```

Website akan live di: `https://username.github.io`
